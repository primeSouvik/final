Projects Undertaken

#1: Operating things via Internet

In this case we created system that can be used to control verious electro-mechanical devices like fan, tv, motor,light via internet by a website.Using NodeMCU and php based server we achived our goal

#2: Weather monitoring system and prediction

For wather monitoring system we again used NodeMCU and verious sensors like LM35(temperature), DHT11(Humidity), LDR(light) Uploading data from NodeMCU to server continously after some interval and storing those values into CSV file. Also a graph is included for visualization purpose. We used Machine-learning algorithms to predict wather.
